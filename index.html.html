Vamos começar

Primeiramente vamos explicar, o que HTML? O HTLM é uma linguagem de hipertexto e marcação, e esta é sua principal característica, o que a torna tão poderosa para desenvolvimento de sites. As marcações são inseridas nos textos, porém para os olhos dos leitores elas são invisíveis, pois sua principal finalidade é de passar instruções ao navegador para que o mesmo possa renderizar uma página. Essas intruções são criadas pelas Tags, palavras que tem o propósito de dar alguma ordem ao navegador para que ele possa renderizar corretamente as linhas de códigos




!DOCTYPE html

É usado para orientar que o arquivo está na liguagem HTML





html

O < html > é a Tag que ínicia um aquivo em HTML, sendo somente precedida pelo < !DOCTYPE > e todos outro elementos HTMLs ficam dentro dele





head

Essa tag é a que contem os meta dados, nela adicionamos todas a informações que seram uteis, mas não visiveis no site (somente pela ferramenta de desenvolvedor)





title

Essa tag é usada para escrever titulos, no caso, ela é quem nomeia a guia do site, como a deste, que está "SITE PARA APRESENTAÇÃO DE TAGS"





body

A Tag < body > é a primeira parte que ínicia aonde os usuários veêm, nela está contido, todo texto e informação visual que vemos na página





h1 to h2

Esse conjunto de Tags, que vão desde o h1 até o h6, servem para que possamos destacar palavras e textos, exemplo:

EXEMPLO

EXEMPLO

EXEMPLO





p

A Tag < p >, é usada para escrever paragráfos, por exemplo, essa frase foi escrita com a mesma





br

A Tag < br > é utilazada para pular um texto, ou um conjunto de informações para a próxima linha, exemplo:

ESSE TEXTO FOI
PULADO PELA TAG





hr

A Tag < hr > é utilazada para que se possa criar uma linha, como no exemplo abaixo, isso serve por exemplo para que se possa dividir conteúdos





!--...--

A Tag < !--...-- > é utilazada para fazer um comentário, essa tag é mais para que outros programadores quando estejam olhando o código fonte, possam entender a linha de raciocínio utilizada para aquele parte em específico





meta

A Tag < meta > é uma Tag não visível ao público, porém de extrema importância, pois nela são contidas meta dados que possibilitam o site funcinar corretamente, alguns desses atributos são:

Charset
Content
Name
Http-equiv

Esses ainda, podem adquirir um conteudo, por exemplo, "Name" pode receber valor de autor, descrição, palavras-chaves ou ainda a porta do site. Já no caso do Charset, ele é uma ferramenta utilizada para indicar ao navegador, qual conjunto de caracteres deveram ser renderizados e utilizados na página web em questão.





base

Essa tag é utilizada para incicar a base dos URL, ou seja, ela indica a raiz dos link da página, para que seja mais fácil o direcionamento, assim ao clicar em um link, a pessoa é direcionada corretamente para um conteúdo específico dentro de uma grande plataforma por exemplo





basefront

Essa tag não é mais suportada pelo HTML5 ela era utilizado para as partes visuais do site, ela caracterizava, cor, tamanho ou fonte, dos textos, atualmente é mais viável ser utilizado uma segunda linguagem de programação que complementa o HTML, chamada CSS
